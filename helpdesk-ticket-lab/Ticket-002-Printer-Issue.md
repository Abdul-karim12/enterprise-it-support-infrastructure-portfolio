# Ticket 002 - Printer Not Printing

## Ticket Information

**Ticket ID:** HD-002
**Priority:** Low
**Category:** Hardware Support
**Status:** Resolved

---

## User Issue

User reported that documents were stuck in the print queue and would not print.

---

## Troubleshooting Steps

1. Verified printer power and network connection.
2. Checked Windows printer status.
3. Cleared print queue.
4. Restarted Print Spooler service.
5. Removed and re-added printer.
6. Sent test print.
7. User confirmed printer was working.

---

## Root Cause

Corrupted print queue prevented jobs from processing.

---

## Resolution

Cleared queue and restarted print services.
